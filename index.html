使用 Hexo架部落格

Hexo 是一個快速、簡單且強大的網誌框架。Hexo 使用 Markdown（或其他渲染引擎）解析您的文章，並在幾秒鐘內，透過漂亮的主題產生靜態檔案。

安裝 Hexo

安裝需求

安裝 Hexo 相當簡單；然而，在安裝前您必須先檢查下列您的電腦是否已經安裝下列軟體：

  1. Node.js
  2. Git

安裝 Hexo

若您的電腦已經安裝上述的必備軟體，那麼恭喜您！只需要透過 npm 即可完成 Hexo 的安裝。

More info: 官網更詳盡的安裝說明

開始建置 Hexo Blog

一旦完成Hexo的安裝後，請執行以下指令，Hexo會在指定的資料夾中建立所有需要的檔案

建置完成後，專案資料夾中會有下列檔案:

在這邊只先介紹指個目前所需要了解的檔案和資料夾

  1. _config.yml
    網站配置檔案，可以在此配置大部分的設定

  2. source
    此資料夾為原始檔案資料夾，是放置Blog內容的地方。
    檔案和資料夾名稱開頭若為 _ (底線) 和隱藏檔案會被系統忽略，除了 _posts 資料夾以外，而其中的Markdown 和 HTML 檔案會被系統處理並放到 public 資料夾，而其他檔案會被拷貝過去。

More info: 官網更多建置相關的介紹

Hexo Blog 本地端測試

以上步驟就已完成建置了，接下來只要輸入以下指令，就可以在本地電腦端看到Blog

然後打開瀏覽器，輸入http://localhost:4000 就可以看到你的Blog囉!

設定git

設定電子郵件和名子

# git config --global user.email "you@example.com"
# git config --global user.name "Your Name"

常用的Alias

$ git config --global alias.co checkout
$ git config --global alias.br branch
$ git config --global alias.ci commit
$ git config --global alias.st status
$ git config --global alias.last 'log -1 HEAD'

beatiful log

$ git config --global alias.lg1 "log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold green)(%ar)%C(reset) %C(white)%s%C(reset) %C(dim white)- %an%C(reset)%C(bold yellow)%d%C(reset)' --all"
$ git config --global alias.lg2 "lg2 = log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold cyan)%aD%C(reset) %C(bold green)(%ar)%C(reset)%C(bold yellow)%d%C(reset)%n'' %C(white)%s%C(reset) %C(dim white)- %an%C(reset)' --all"
$ git config --global alias.lg 'lg = !"git lg1"'
