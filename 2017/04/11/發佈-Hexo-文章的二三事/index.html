發佈 Hexo 文章的二三事

發表於 2017-04-11 | 分類於 Hexo

在完成建立 Hexo Blog 之後，您一定迫不及待想要發佈自己的文章了吧! 以下就讓小編介紹一下與發布文章相關的二三事吧 :)

一樣分成三個步驟：

  1. 創建一篇新文章
  2. 認識佈局、模板和 Front-matter

  3. 撰寫文章與檢視

第二步：認識佈局、模板和 Front-matter

佈局 (Layout)

Hexo 有三種默認布局： post、page 和draft，他們產生出來的文建會分別存在不同對應的路徑：

布局 路徑
page source
post source/_posts
draft source/_drafts

而若您有自己定義的其他布局，則會和 post 相同，都將儲存到 source/_posts 的資料夾中。

模板 (Scaffold 或稱鷹架)

在建立新文件時， Hexo 其實是根據 scaffolds 資料夾中相對應的模板來建立文件的，例如最常用的 post：

Front-matter (前言事物)

Front-matter 是檔案最上方以 --- 分隔的區域，用於指定個別檔案的變數，舉例來說：

以下是預先定義的參數，您可以在模板中取得這些設定並加以利用

參數 描述 備註
layout 佈局
title 標題
date 建立日期 檔案建立日期
updated 更新日期 檔案更新日期
comments 開啟文章的留言功能 true
categories 分類（不適用於分頁） 有順序和階層性的
tags 標籤（不適用於分頁） 沒有順序和階層性的
permalink 覆蓋文章網址 (一般用不到)
