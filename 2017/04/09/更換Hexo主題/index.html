更換 Hexo 主題

發表於 2017-04-09 | 分類於 Hexo

在完成佈署 Hexo Blog 後，您可能會發現官方默認的主題太醜 XD 或是想要挑個符合您品味的主題，所以接下來就要介紹如何更換 Hexo Blog 主題。

大致可以分成三個步驟:

  1. 選擇主題
  2. 安裝主題
  3. 啟用與驗證

第一步：選擇主題

大概有以下兩種方式尋找自己想要的主題:

  1. 官方提供的主題頁面
  2. 在github 搜尋 hexo-theme

而在這邊就選擇 github 排名第一的主題 NexT 來做示範

第二步：安裝主題

在此提供兩種方式安裝主題

  1. 使用 clone 的方式:
    如果您對於 Git 有初步的了解，這是比較推薦的方式
  2. 使用下載壓縮檔案的方式:
    假如您對於 Git 不熟悉的話，請使用此方法

使用 clone 的方式安裝主題

此方式的好處是之後的更新就可以通過 git pull 來快速更新，而不用再次下載壓縮包替換，輸入以下指令就可以完成安裝囉：

$ cd <your-hexo-code-folder>
$ git clone https://github.com/iissnan/hexo-theme-next themes/next

若遇到最新版的程式有問題，可以去 release 發布頁面 查看您所需要的版本號 (或是直接看 log)，並用使用 checkout 切換到穩定的版本，例如 v5.1.1

$ cd <your-hexo-code-folder>
$ cd themes/next
$ git checkout v5.1.1

使用下載壓縮檔案的方式安裝主題:

  1. 下載所需要的版本

    請前往 release 發布頁面 中選擇您所需要的的版本

    並點選 Downloads 中的 source code 來下載，並解壓縮。

  2. 解壓縮後放到 themes 資料夾中

    接著把解壓縮後的 hexo-theme-next-5.1.1 資料夾改名稱為 next ，並放到 Blog 專案中的 themes 資料夾中，即可完成安裝

第三步：啟用與驗證主題

開啟 _config.yml 配置文件，在 theme: 字後輸入 next，如:

theme: next

接著就可以在 terminal 回到 Blog 專案的根目錄，並輸入指令來驗證主題是否運作正常

$ hexo s --debug # 可以觀察是否有異常信息

訪問 http://localhost:4000/Blog 來檢視我們新安裝的主題囉!
