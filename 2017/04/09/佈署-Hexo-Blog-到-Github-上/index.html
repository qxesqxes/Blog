佈署 Hexo Blog 到 Github 上

發表於 2017-04-09 | 分類於 Hexo

在完成建置 Hexo Blog 後，您一定不滿足只能在本地端看到自己的 Blog 吧，所以接下來要介紹要如何把本地端建置好的 Hexo Blog 佈署到 Github 上囉!

一樣幫您分成簡單的三個步驟：

  1. 在 Github 新增一個 repository
  2. 安裝 hexo-deployer-git 和設定配置文件
  3. 完成佈署 Hexo Blog

回到到 terminal 輸入:

設定配置文件 _config.yml

設定URL的部分

url: http://qxesqxes.github.io/Blog
root: /Blog/

設定deploy的部分

deploy:
      type: git
      repository: https://github.com/qxesqxes/Blog.git
      branch: gh-pages
